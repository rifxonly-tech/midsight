How it works

For logged-in users, chat history is saved. Guests can try mediSight once without account creation.

1. Landing → Login Click “Try MediSight AI” and choose to sign in or continue as guest.
2. Camera detection The browser opens the camera, Laravel calls the Python inference_example.py script, and receives JSON output.
3. AI Chat The chatbot uses those values as initial context for the conversation.

Privacy & testing first

Before integrating, your teammate should test Python manually in a terminal. Once the JSON is correct, Laravel can safely call it in the background for every “Detect” request.
